*投票の練習 (?) [#q194f183]

全く意味のないサザエさん投票です。これは投票のテストです。

#vote(カツオ[0],タマ[0],サザエ[0],マスオ[0],タラオ[0],波平[0],フネ[0],ワカメ[0],アナゴさん[0],伊佐坂先生[0],ウキエさん[0],ハチ[0],花沢さん[0],花沢さんの父[0],三河屋のサブちゃん[0],ノリスケ[0],タイ子さん[0],イクラ[0],中島[0],海平[0])

と書くと、投票を設置できます。ただし、行頭にはスペースなどを含まず、必ず「#」ではじめてください。~

*BASICプログラムの記述 [#mc3c1296]

 #basic{{
 PRINT "こんにちはこんにちは!"
 }}

と書くと、BASICプログラムをプチコンフォントで記述できます。

#basic{{
PRINT "こんにちはこんにちは!"
}}

 &basic{ RUN };

と書くと、&basic{ RUN }; こんな感じで文章中にも使えます。
