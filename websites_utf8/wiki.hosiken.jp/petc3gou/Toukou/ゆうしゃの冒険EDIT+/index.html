Hello!! ゆうしゃの冒険 worldへ!


2016年5月12日投稿 / 投稿者 : MGウッディー / タグ : 投稿プログラム ゲーム 横スクロールアクション ステージ作成

※本ページ内の画像は、Miiverseで投稿されていた画像を使用させていただきました。

MGウッディー作 ゲーム一覧

(一作目) ゆうしゃの冒険
  • 2014年末に公開された記念すべき第一作。主人公である「ゆうしゃ」のキャラクターの外観、操作性は特に変化はなく今作でも健在。ステージ作成機能はver3.0で組み込まれた。
    詳細はこちら
(二作目) ゆうしゃの冒険 DX
  • 一作目のver5.0として公開する予定が、相違点が多いと自負したため、思い切ってDX版として公開。正直相違点はそれほどない。
    詳細はこちら
(三作目) ゆうしゃの冒険 EDIT+ (本作)
  • 一から新たにプログラムを組み直し、細部までこだわり抜いて2016年5月に完成。ステージ作成の操作性も飛躍的に向上し、利便性も増した。
ニンジャアクション
  • プチコン公式サイトのコンテストに応募するつもりの作品が、応募締切に間に合わずに断念し、Wiki内で公開。忍者型のキャラを箱庭型のステージで動き回り、緑色の巻物を全て集めればクリア。高難易度のゲームになっている。 詳細はこちら

このゲームとは？

  • さて、ゆうしゃの冒険EDIT+とは、前作ゆうしゃの冒険DXの一部の要素は受け継ぎつつ、プログラムは新たに書き直した完全新作です。本ゲームはストーリーモードが無い代わりにEDITモードがかなり使いやすくなっています。前作の仕掛けはもちろん、新しい仕掛けが追加されているので、前作を遊んだことが無い人も楽しく遊べると思います。

遊び方

タイトル画面
  • スライドパッド：ゆうしゃの移動
  • 十字キー：項目の選択
  • ABXYボタン：項目の決定
  • LRボタン：設定の保存
ステージを作るとき
  • スライドパッド：ステージのスクロール
  • Aボタン：エディットメニューの表示
  • Bボタン：グリッドの表示・非表示
  • Yボタン：消しゴムモードの切り替え
  • Xボタン：コピーモードの切り替え
  • LRボタン：オブジェクトのページ切り替え
  • ZL･ZRボタン：操作パネルの表示・非表示
ステージを遊ぶとき
  • スライドパッド：ゆうしゃの移動
  • 十字ボタン：ゆうしゃの移動
  • Bボタン：ゆうしゃがジャンプ
  • Yボタン：ゆうしゃがダッシュ
  • LRボタン：ポーズ画面の表示・非表示 ※ジャンプ・ダッシュの操作の割り当ては、設定から変更できます。

遊ぶときの注意

  • ステージを作る際、２×２マスのオブジェクトの位置を移動させるときは、左上の辺りをタッチすれば上手く持てます。
  • ステージで遊ぶモードでステージを遊ぶと、制作中のステージは遊んだステージに上書きされてしまうため、制作中のステージは保存しておくことをおすすめします。

更新版(ver2.0)の内容
  • 主な更新内容(ver4.0にも含まれます)
    1. バグ修正
      1. SPCOLの書式が間違っていた問題を修正
      2. 隠しブロックを叩くと、稀にミスになるのを修正
      3. スターコインの使用ポイント数が、間違っていた問題を修正
      4. 『げんぎいのじかん』の誤字修正
      5. BGMが稀に二重になる問題を修正
      6. BGの当たり判定を修正
      7. かたいブロックをシェイクするとイチゴになるバグ修正
    2. 動作速度の改善
    3. ちくわブロックを追加
    4. リフトの追加
    5. 音符ブロックがより跳ねやすくなりました
    6. タイトル画面にスライム追加
    7. BGM選曲機能の追加
更新版(ver2.5)の内容
  • 主な更新内容(ver4.0にも含まれます)
    1. 土管連結器をコピーする際、左上部分以外をタッチすると、スライムがコピーされるバグを修正
    2. 動作速度の改善
    3. 当たり判定に関する不具合を修正
    4. 氷ブロックに関連するバグを修正
    5. カッター(敵)に関するバグを修正
    6. ドアが稀に入れなくなるバグを修正など
更新版(ver3.0)の内容
  • 主な更新内容 (ver4.0にも含まれます)
    1. バネの追加
    2. バグ修正
    3. 動作速度の改善
    4. その他色々追加(遊んでからのお楽しみ)
更新版(ver4.0)の内容
  • 主な更新内容
    1. レールに動かせるものが増えました(レンガブロック、ファイアバーなど)
    2. 身体障がい者向けサポート機能を追加
    3. 大幅な動作速度の改善
    4. 大幅なバグ修正 など
修正版(ver4.25)の内容
  • 主な更新内容
    1. バグの修正
