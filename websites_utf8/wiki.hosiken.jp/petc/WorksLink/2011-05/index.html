プログラムリストあり

動画中でプログラムリストが紹介されている作品です。

2011/05/01 : SPLIT!

  • 2 ちゃんねる プチコンスレ その4 >>100 より。
  • 分裂型UFOの来襲を誘爆ミサイルで迎撃するシューティングゲーム。誘爆すると花火みたいでキレイ。
  • プログラムリストは約 160 行。

2011/05/02 : タッチペンを使った STG の試作

  • タッチペンで自機のまわりのオプションを回転させることができる、シューティングゲームの試作。
  • 作者さんのブログに解説があります。

2011/05/06 : マスターマインド

  • 2 ちゃんねる プチコンスレ その4 >>239 より。
  • 昔懐かしいボードゲーム。4つ並んだ4〜8色の駒を推理して当てていく。
  • プログラムリストは 160 行。

2011/05/10 : トランプゲーム開発用素材

  • トランプゲームではなく、ゲーム作成に必要な処理共通ルーチン。
  • QRコードはこちら。

2011/05/18 : PUSH ROAD

  • 倉庫番風ゲーム。主人公を出口まで導く。
  • 作者ブログにＱＲコードとプログラムリストがある。
  • プログラムリストは 375 行。

  • サクサクバージョンもあり。
  • 動作がサクサクになったのではなく、tvk の sakusaku で紹介されたバージョンのようです。
  • キャラがおねーさんと白井ヴィンセントに変更されています。

2011/05/18 : MISSILE DEFFENCE

  • ミサイルコマンド風ゲーム。
  • 作者ブログに解説がある。プログラムリストはこちら。
  • プログラムリストは 596 行。

2011/05/18 : サウンドテストツール

  • BGMやBEEPを能率よく連続して聞くためのツール。プログラム作成中の選曲に便利。
  • QRコードはこちら。

2011/05/27 : スペースマウス

  • 芸夢狂人氏作のパソコン用ゲームの移植。敵を避けて 250 階までのぼって行く。
  • QRコードはこちら。
  • 原作の動画はこちら。
  • さらにキャラデザインなどにアレンジを加えたPCGスペースマウスもあり。

2011/05/29 : 自前演奏ルーチンで BGM を鳴らすテスト

  • シンプルな BGM 演奏ルーチンのデモプログラム。ゲーム部分が処理落ちしても演奏のテンポを維持できる。

2011/05/31 : プチ 3D

  • 三角錐や傘がが回転するプログラム。
  • 上下画面を左右の目で見ることで、自力での裸眼立体視ができる…はず。

プログラムリストなし

動画中でプログラムリストが紹介されていない作品です。

2011/05/05 : テトリスを作ってみる その3

  • テトリスの完成版。SRS、T-SPIN、BtoB、MML 実装。

2011/05/05 : D.P.S.

  • ディープスキャン風の潜水艦ゲーム。

2011/05/05 : マインスイーパを作ってみた

  • タッチペンを使用したマインスイーパ。上級者モードも搭載。

2011/05/11 : THE 医者 (未完成)

  • 医者シミュレーションゲーム。
  • 薬品の在庫管理や医者本人の体調管理をしながら、患者を診療していく。

2011/05/12 : Action Games

  • 鍵を取って出口まで向かうジャンプアクションゲーム。
  • 横浜在住の英語・日本語バイリンガルの方が制作。

2011/05/12 : プチコン RPG

  • プチコンで RPG っぽいものを作ろうという試み。
  • Action Games と同じ作者で英語のナレーションが入っているが、作品内ではカタカナを駆使している。

2011/05/14 : Mr.Do! もどき

  • 大昔の名作アーケードゲームの移植。脅威の再現度!!

2011/05/19 : カニーバル

  • 大昔のセガのアーケードゲームを再現。
  • 射的ゲームで、弾が切れると終了。

2011/05/19 : 探索型アクションゲームを作る

  • ガリウスの迷宮やグーニーズIIのような探索型アクションゲーム。

2011/05/21 : 探査機はやぶさのゲーム

  • はやぶさが小惑星イトカワまで行き、サンプルをとって地球に帰ってくるまでをゲーム化。
  • 動画説明に詳しい解説が書かれています。

2011/05/24 : デジタルインベーダー

  • 昔流行したゲーム電卓 (ゲーム機能付き電卓) に搭載されていたゲームのひとつの移植。
