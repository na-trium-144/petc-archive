BASICプログラムの記述

#basic{{
PRINT "こんにちはこんにちは!"
}}

と書くと、BASICプログラムをプチコンフォントで記述できます。

  1. ＰＲＩＮＴ　”こんにちはこんにちは！”
&basic{ RUN };

と書くと、 RUN こんな感じで文章中にも使えます。
