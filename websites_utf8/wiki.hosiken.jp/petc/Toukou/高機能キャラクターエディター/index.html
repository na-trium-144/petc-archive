2012年4月18日投稿 / 投稿者 : uxtuno_ / タグ : 投稿プログラム 実用ツール QRコード mkII専用

概要

ＵＸ＿ＣＨＲＥＤ　Ｖｅｒ１．５です。

プチコン付属のＣＨＲＥＤよりは高機能を目指して作りました。
致命的なバグがあるかもしれないので使用は自己責任で。
アンドゥ、リドゥの操作はDSiウェアのうごメモを意識していたりします。
QRコードが読み取りにくかったらこっちでダウンロードしたらいいかもしれません。

http://ux.getuploader.com/petitcom_mkII/download/41/

操作説明

十字キーカーソル移動（上画面）
Ａボタン選択位置（上画面カーソルの左上）から４キャラ分定義情報をファンクションキーに登録
Ｂボタンカーソルモード（十字キーで移動、Ａボタンで点を打つモード）に変更
Ｙ＋左アンドゥ
Ｙ＋右リドゥ
Ｙ＋上ペースト
Ｙ＋下コピー
Ｙ＋ＬＲ簡易パレット変更
（Ｙ＋Ｘ）＋上重ねペースト
（Ｙ＋Ｘ）＋下重ねコピー
ＬＲコマンドタブ切り替え

主な機能

  • アンドゥ、リドゥ（7回まで）
  • コピーペースト
  • ＳＰＳ、ＳＰＤなども読み込める
  • カーソルサイズとキャラの並びを別々に設定できる
  • 文字列としてファンクションキーに登録
  • スクロール
  • ＨＳＢでの色変更
  • パレットコピー、ペースト
  • 色変更をＣＨＲＥＤより直感的に
  • 「！」で保存すると上書き保存
    などです

コマンドモード説明

スタートボタンを押すとコマンドモードになります。
最初にコマンドを入力してからパラメータという感じです
エラー処理はほとんどしてないので注意。

Q コマンド

パラメータ

無し

説明

コマンドモードを終了します。

CHRCOPY コマンド

パラメータ

転送元キャラバンク名,開始位置,数,転送先位置

説明

転送元から指定数のキャラクタを表示中のバンクへコピーします
例）SPU1,0,256,0
これはSPU1の定義情報をそのまま現在表示中のバンクへコピーします。

COLCOPY コマンド

パラメータ

転送元色バンク名,開始色番号,数,転送先色番号

説明

転送元から指定数の色を表示中のバンクの色へコピーします
例）BG,16,16,0
これはBGの色を１６番から１６個表示中のバンクの色へコピーします。

COLSET コマンド

パラメータ

色番号,色情報文字列

説明

色を数値指定できます。
例）2,00FF00
これは表示中のバンクの２番の色へ緑を登録します

COLREVSET コマンド

パラメータ

色番号,色情報文字列

説明

数値指定した色の補色を登録します。
例）2,00FF00
これは表示中のバンクの２番の色へ緑の補色を登録します

TINT コマンド

パラメータ

開始色番号,色数,色情報文字列

説明

指定範囲の色の色合いを変化させます。
例）0,16,FFFFFF
これは表示中のバンクの色の０〜１５を白っぽい色に変化させています

MODE コマンド

パラメータ

表示モード（０〜２）

説明
  • ０＝通常
  • １＝３バンク同時表示（コピーペースト系の操作は使えなくなります）
  • ２＝操作説明の所に各パレットのプレビューを表示
