** 使い方 [#ja77f5e8]
打ち込んだら、&basic{{RUN}}してください。そうすると、カーソルがちっかちっか点滅しているはずです。そこでなにか(例えば&basic{{テースートーDAYO}}とか。)打ち込んで下さい。仕上げにenter押せば、もう保存できちゃいました。終わるときは空っぽにしてenter押せば終わります。

* プログラムリスト [#g0af827b]
#basic{{
GCLS 0:CLEAR:DIM A$(1529)
FOR I=0 TO 1528
LINPUT A$(I)
IF A$(I)=="" THEN END
FOR U=0 TO LEN(A$(I))
B$=MID$(A$(I),U,1)+A$
IF B$==A$ THEN U=LEN(A$(I):NEXT:NEXT
GPAGE 1
GPSET OLDX,OLDY,ASC(B$)
OLDX=OLDX+1
IF OLDX>255 THEN OLDX=0:?"PPP":OLDY=OLDY+1
NEXT:NEXT
}}
