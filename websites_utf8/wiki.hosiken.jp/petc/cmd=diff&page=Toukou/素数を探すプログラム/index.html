* プログラムリスト [#c662ffe4]
#basic{{
CLS:VISIBLE 1,1,0,0,0,0
CLEAR:INPUT"ﾄﾞｺﾏﾃﾞｻｶﾞｼﾏｽｶ";M
A=1S=2FOR I=0TO 1I=0A=A+1
R=S>M:IF R THEN?C"ｺﾐﾂｹﾀﾖ":END
R=A>S/A:C$=STR$(S)+","
IF CSRX+LEN(C$)>32THEN?
IF R THEN A=1?S",";:C=C+1
IF R THEN S=S+1NEXT
R=S/A%1==0 AND S/A!=0
IF R THEN A=1S=S+1
NEXT
}}
