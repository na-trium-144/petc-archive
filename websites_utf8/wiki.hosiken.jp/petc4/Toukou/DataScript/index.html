概要

data 文でスクリプトを書いて実行できるライブラリです。シナリオスクリプトに向いているのではないかと思います。

data 文で書いたスクリプトをテキストファイルとして保存して、自分の作品から呼び出して使うという形です。

SmileBasic と違い、並列的に実行ができます。プロジェクト内にあるサンプルのうち、TIC.DSC はその使い方を示しています。

以下は最も簡単な、ハローワールド的スクリプトです。

  1. ｄａｔａ　＃ｐｒｉｎｔｌｎ，”Ｈｅｌｌｏ　ＤａｔａＳｃｒｉｐｔ”

DataScript の実行

まず Process_Start で実行をスタートさせます。引数には実行する場所を指定します。任意の @ ではじまる場所か最初から実行するならば @_Start を指定します。

いずれの場合も、スロットの指定を忘れないようにして下さい。

リファレンス

#end : スクリプトを終了

data #end

この位置でスクリプトを終了します。ただし、スクリプトの最後では自動的に終了するため、このコマンドは必須ではありません。

#goto : 特定の場所にジャンプ

data #goto, @label

  • @label : ジャンプ先の label

指定した @label という label にジャンプします。無限ループを作る時もこのコマンドを使います。

ジャンプ先の label は data をつけず SmileBasic と同じように書きます。

#call : サブルーチンを呼び出す

data #call, @label [, ...]

  • @label : サブルーチン
  • ... : サブルーチンに渡す引数

サブルーチンを呼び出します。#goto とことなるのは点は、ジャンプ先の #return でここに復帰できることと、任意で引数を渡すことができる点です。

サブルーチン側では #param コマンドで渡された引数を取得できます。

また同じ使い方の #start を使うと、新しいプロセスを作成してサブルーチンを非同期で呼び出すことができます。サブルーチンをはじめた後は直ちに #start コマンドの直後のコマンドを実行します。
