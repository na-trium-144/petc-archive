* いらっしゃいませ [#xff5b894]

#basic(3gou){{
'── なんか それっぽい プログラム ──
CLS : CL=0 : Z=0
WHILE 1
  INC Z : IF Z>200 THEN Z=0
  FOR I=0 TO 14
    LOCATE 9,4+I,Z : COLOR (CL+I) MOD 16
    IF I MOD 3==0 THEN PRINT "┏━━━━━┓"
    IF I MOD 3==1 THEN PRINT "┃　れいわ　┃"
    IF I MOD 3==2 THEN PRINT "┗━━━━━┛"
    IF I MOD 3==0 THEN PRINT "┏━━━━━━━━━━┓"
    IF I MOD 3==1 THEN PRINT "┃　2020あけおめ　┃"
    IF I MOD 3==2 THEN PRINT "┗━━━━━━━━━━┛"
  NEXT
  CL=(CL+1) MOD 16 : VSYNC 2
WEND
}}

こちらは、ニンテンドー 3DS ダウンロードソフトの BASIC インタプリタ『[[プチコン3号 SMILE BASIC:http://smileboom.com/special/petitcom/]]』、およびそれの Wii U 版である『[[プチコンBIG:https://www.nintendo.co.jp/titles/20010000024627/]]』のまとめ Wiki です。ここでは、上記のようにプチコンの BASIC プログラムを簡単に書けるようになっていたり、アップロードした作品を紹介したりできます。おっさんも大人も子供もおねーさんも集え。

プチコンの公開キーを投稿したい場合、ひとつの作品として成立しているものであれば、ぜひ[[投稿プログラムコーナー>Toukou]]へ投稿してみてください。作りかけのものや、テストプログラムなどの公開キーを気軽に見せたい場合は[[OFコーナー>OverFlow]]や[[Petitverse>Board]]に書き込んでみてください。

Wiki にプログラムリストを書く際の書式や、編集のガイドラインなどは[[ヘルプ>Help]]のページを見てください。
