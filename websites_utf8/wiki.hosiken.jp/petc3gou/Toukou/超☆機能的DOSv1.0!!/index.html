2020年1月6日投稿 / 投稿者 : Endergreen / タグ : 投稿プログラム 1画面プログラム 実用ツール

概要

初心者が作ったDOSv0.7.5がアップデートし、v1.0完成!! 漢字化や新機能が追加!! とても実用的!!

動作画面

機能

このプログラムでHELPと入力すれば表示されます (このプログラムで出てくる情報は本当です!!)
DIR
TIME
DATE
VER
COPY
DELETE(DEL)
RENAME(REN)
SYSTEMINFO
このプログラムは単体で動くため、他のフォルダに移動しても動作します!
さらになんと、このプログラムは、 コマンドを追加 出来ちゃうんです! やり方:

  1. ＩＦ　ＣＭＤ＄＝＝コマンド名　ＴＨＥＮ
  2. 実行する事
  3. ＧＯＴＯ　＠１
  4. ＥＮＤＩＦ

をプログラムの中のENDIFの次に書いてください!

注意

だいぶ再現しているのですが、限界があるため、del **.txtなど打つときも、DELを入力し、エンターを押してファイル名を入力してください。

更新予定

コマンドはこれからも増やしていく予定です!!
まだ技術がないので無理ですが、.bat(バッチファイル)に対応させたいです!ファイルに書いてあるコマンドを実行する的なやり方を知っている人は教えて下さい!
