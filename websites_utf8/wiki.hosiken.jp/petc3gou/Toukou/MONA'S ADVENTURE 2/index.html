ストーリー

モナーとギコの活躍により、モララーに捕まったショボーンは助け出されました。
村に帰ったモナー達は、しぃから大変なことを知らされます。
"クックルドゥドゥドゥ"が村に現れ、罪のない人々(?)を殴っているのです。
モナーとギコに、前作の敵のモララー、そしてしぃも加わり、
クックルのいる"エリア51"を目指して旅立ちます…

(ストーリー分かりにくくてすみません…)

ゲーム内容

「モナー」「ギコ」「モララー」「しぃ」を操作して6つのステージを攻略するジャンプアクションゲームです。
基本的な仕様は前作と同じですが、難易度が高くなっています。

操作方法

  • 3DS/ゲームパッド
    • 十字キー左右 : 移動
    • 十字キー下 : しゃがむ
    • Aボタン : ジャンプ
    • Bボタン : 通常攻撃
    • Xボタン : 必殺技(緑ゲージ消費)
    • Rボタン : ポーズ
  • Wiiリモコン
    • 十字キー左右 : 移動
    • 十字キー下 : しゃがむ
    • 2ボタン : ジャンプ
    • 1ボタン : 通常攻撃
    • Bボタン : 必殺技(緑ゲージ消費)
    • +ボタン : ポーズ

操作キャラクター

モナー
ジャンプ力、走る速さ共に平均的で、使いやすいキャラです。
攻撃は光弾。離れた敵を攻撃できます。
ギコ
ジャンプ力は一番ですが、走りはモナーに若干劣ります。
攻撃はヨーヨー。十字キーと組み合わせて、真下を除く7方向に打ち分けできます。
モララー
ジャンプ力は最低ですが、走りは一番速いです。
攻撃は剣。隙の小さい攻撃ができます。
しぃ
二段跳びができますが、走りは一番遅いです。
攻撃は手榴弾。一発で大きなダメージを与えます。

アイテム

回復ドリンクP
体力を3目盛り回復します。
回復ドリンクS
体力を全て回復します。
1UP
残機が1つ増えます。
特定のキャラでないと取れないことが多いです。

攻略のコツ
+  攻略のコツ
  1. 敵の行動パターンを覚えてしまおう！
  2. 軟らかい敵だけを相手にし、堅い敵はなるべく無視しよう！
  3. 一部の敵の弾は、攻撃を当てて破壊したり、跳ね返したりできるぞ！
  4. 氷の上では、ジャンプするとすぐに止まれる！
  5. やられると必殺ゲージは空になってしまうので、
    ピンチになったら惜しげなく必殺技を使おう！
  6. 諦めずに立ち向かえば、必ず奴を倒せる！ 頑張れ！
