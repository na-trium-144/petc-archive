2013年6月25日投稿 / 投稿者 : 猫姫 / タグ : 投稿プログラム ゲーム mkII専用

更新情報2013/06/25

　現在、最新バージョン[VER1.10]で公開しています　

即日で申し訳ありません、重大なミスを発見したので、VERUPしました。

PRG:NHPSPS
  • 連続でゲームをすると、図柄の配置が変わらないバグを修正しました。
  • リザルトで表示されるランキングのずれを修正しました。

内容

神経衰弱ゲームです。
最初に図柄の配置を覚えて、その後、伏せられた絵柄をそろえます。
覚える時間、解答時間共に得点に反映されるので、
早く覚えて、早く解く。
脳トレ的なゲームです。

遊び方

QRコード

  • PRG:NHPSPS
  • CHR:NHPSPS
  • COL:NHPSPS
    ３つのリソースがあるので、すべて読み込んでください。

メニュー画面

操作はタッチペンで行います。
下画面、4つのメニューから、選んでください。

  • GAME START
    ゲームを開始します。
  • SCORE RANKING
    今までの記録TOP30を表示します。
  • DELETE MEM FILE
    セーブデータを削除して、PRGを終了します。
  • SAVE & GAME END
    ランキングをセーブして、PRGを終了します。

ゲームの流れ

  • GAME STARTをタッチすると、しばらくのINITIALIZEを経て、下画面に「TOUCH START」と表示されます。
  • 下画面をタッチすると、4種類16枚の図柄が出るので、覚えてください。
  • 覚えたら、DSの蓋を閉じて、しばらく待ってください
  • DSを開くと、図柄が伏せられた状態になっています。タッチペンで図柄をタッチして、絵合わせをしてください。
  • 16面すべてを合わせるとクリアです。ゲームのリザルトが表示されます。

製作ノート

+  ネタバレ注意

コンセプト

発端は、半年前に作ってあった、「HIT 3 NUMBERS」を、公開できるように作り替えた際に、プチコン技を色々理解した今なら、何かできるんじゃないかな。といったところにあり、さらに、

  • 公式素材を使ってみたかったんじゃ!!
    ということで、スマイルブームプレゼント素材を、尽くDLしてきました。
    そして、これも実は再現だけはしてあるのですが、スマホアプリの数字タッチゲーム。
    あの操作感覚でできるゲームを作ろう。
  • 絵合わせでどうだろう
    と、いうことで、
    「タッチパネルを素早くタッチしてタイムを競う神経衰弱」
    になりました。

chrデータの編集

公式素材のハカセ達は。それぞれパレットNOが違うので、chrデータと共に、パレットのデータもいじってます。
まったく、これだけのためにQRが増えていくのは哀しい感じがします。
顔のデータだけをコピーして、顔データに使われていない色を、枠と背景の色に変えてあります。

絵柄のマスク

絵柄と、それを隠しているえ絵柄、それぞれ、別のSPで指定して、重ねています。
こういうと、BGでいいじゃん?てことにもなるんですけどね。
タッチして、絵柄を開く(=上に乗っているマスク用SPを、画面外に飛ばす)と処理しています。

カード開閉の処理

当たり判定

「HIT 3 NUMBERS」のときは、透明のSPを用意して、タッチ座標に移動するようにして、SPHITSPで判定をとってみましたが、いまいちでした。
位置の座標と、判定をとるサイズがわかっているので、X.Yそれぞれを、IF文で判定を出した方が、やはり応答がよかったです。

開閉のフラグ

変数一つ、ビット演算でフラグ処理を行っています。

開発&編集後記

先日UPした、HIT 3 NUMBERSから、今タッチパネルゲームを作るとどんな感じになるかなと、組み始めたプログラムです。
インスピレーションは、スマホ黎明期(?)に流行ったアプリから、
これもテストで組んだことがあるので、今回のパネルの位置取りなどは苦労無くできました。
ランキングは無駄に30位まで用意したのは、ランキングスクロールをやってみたかっただけです。(笑
