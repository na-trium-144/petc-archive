2012年12月1日投稿 / 投稿者 : ショルダー / タグ : 投稿プログラム ゲーム アクションゲーム QRコード mkII専用

概要

アクションゲームです。全５０ステージ有ります。 前回バージョンから、新しい敵キャラ、ギミックを追加しています。

ゲームのルール

  • 主人公は導火線で炎を誘導して敵にぶつけて倒します。
  • 全ての敵を倒すとステージクリアです。
  • 主人公は敵や炎に当ってもゲームオーバにはならないですが、一定時間動けなくなります。
  • 炎は導火線を伝って移動しますが導火線が途切れると勢いが弱くなり最後には消えてしまいます。
  • 画面上の全ての炎が無くなるとミスとなります。
  • 画面上のハートが無い状態でミスになるとゲームオーバです。

操作方法

  • 十字キー：主人公の移動
  • Ａボタン：導火線（通常速度）を置く
  • Ｂボタン：導火線（炎の速度が速くなる）を置く
  • スタート：ポーズをかける

ステージ編集

  • ステージ編集モードを追加しました。
  • ステージデータ作成のおおまかな流れは下記の動画をご覧ください。
  • 自作したデータはGRPリソースとして保存出来ますので、オリジナルデータをQRコードとして公開することも可能です。
ステージ編集の説明

この説明書ではステージ編集モード時の操作方法について説明しています。このモードを利用する場合には、出来る限りプチコンの「プログラムを作る」を選択し、LOAD”SHOHD2”と実行しRUNする事をお奨めします。理由は間違って「SELECT」ボタンを押した場合でも、再度RUNすれば直前まで編集したデータで編集作業を再開出来る為です。

編集メインメニュー

タイトル画面から「ステージノヘンシュウ」を選択すると以下の画面が表示され、各メニューに対応した機能を実行出来ます。各機能の説明は、3項以降を参照下さい。 ※編集モードの初回利用時にのみ「利き手」を聞かれますのでＡ又はＢボタンを押して選択して下さい。（左右のボタンに割りつけられる機能が入れ替わります。）

ファイル（FILE）メニュー
  • A) セーブ（SAVE） 
    編集したステージデータをセーブします。GRP3リソースを使います。
ファイル名指定は下画面にキーボードが表示されますのでこれを使います。
  • B) ロード（LOAD） 
    保存されているステージデータをロードします。GRP3リソースを使います。
ファイル名指定は下画面にキーボードが表示されますのでこれを使います。
  • C) メインメニュー へ モドル（RETURN TO MAIN MENU） 
    タイトル画面に戻ります。
  • D) スベテショッカ（INIT ALL STAGES） 
    現在の編集データ（GRP3リソース）を削除します。この項目選択後に「YES」と
キーボードから入力すると全ステージデータ（５０面分）を初期化（クリア）します。
ステージ（STAGE）メニュー
  • A) ステージセンタク（SELECT STAGE） 
    以下に説明するコピー、削除、挿入や編集するステージを選択します。ステージの範囲は１から５０です。
  • B) ステージコピー（COPY THIS STAGE） 
    現在選択されているステージデータを違うステージ番号にコピーします。
ステージ１番から３番にコピーするには、まずステージ１番を選びます。その後ステージコピーを選んでコピー先の番号を選びます。コピー先のデータは上書きされます。
  • C) ステージサクジョ（DELETE THIS STAGE） 
    現在選択されているステージデータを削除します。ステージ３番を削除するには、まずステージ３番を選んだ後にステージサクジョを実行します。削除実行後はステージ４番以降のデータが１つ前の番号に移動して、ステージ５０番には空のデータが入ります。
  • D) ステージソウニュウ（INSERT A STAGE DATA） 
    現在選択されているステージ番号に一つ分のデータを挿入します。ステージ３番を選択している状態で挿入を実行するとにステージ３番の部分に空データが挿入され、ステージ４番以降のデータが１つ後の番号に移動します。この場合にはステージ５０番のデータは失われます。
  • E) テストプレイ（PLAY THIS STAGE） 
    現在選択されているステージのみをタイトル画面に戻らずにテストプレイ出来ます。
プロパティー（PROPERTY）メニュー
  • A) ヨコサイズ（WIDTH） 
    ステージの横幅を決めます。３２と４０のうちどちらかを選択します。３２の場合はDSの画面サイズと同一となり、画面スクロールしません。４０の場合はステージの幅が大きくなりますが、ゲーム中は画面がスクロールします。デフォルト設定は３２です。
  • B) ＢＧＭ（BGM） 
    ステージのBGMを0〜２９の範囲で設定します。デフォルトで１番のBGMが選択されています。
  • C) カソクスル カンカク（ACCELERATION INT.） 
    プレイ中に一定時間経過すると敵の移動速度が速くなります。この設定で早くなる間隔を調整します。デフォルト値２００（約６０秒）です。設定値は２５５〜１０の範囲です。
  • D) シュツゲン カンカク（MAX）（APPEARANCE INT.(MAX）） 
    ステージに「敵の巣」が配置されている場合の敵が出てくる最大時間です。実際に敵が出てくる間隔は後述の最小時間〜最大時間の間でランダムに決定されます。デフォルト値１００（約３０秒）です。設定値は２５５〜１０の範囲です。設定値１あたり約0.3秒に相当します。
  • E) シュツゲン カンカク（MIN）（APPEARANCE INT.(MIN） ） 
    ステージに「敵の巣」が配置されている場合の敵が出てくる最小時間です。前述の最大時間とペアで使用します。デフォルト値５０（約１５秒）です。設定値は２５５〜１０の範囲です。
ステージ編集時の留意点
  • ステージの外周は侵入出来ない部品（壁や川）で囲んで配置してください。
  • 敵の巣の設置は最大１６個までに抑えてください。
  • 敵や炎はそれぞれ同時３０体までに抑えてください。
  • ジェームス（マイキャラ）は必ず配置してください。
  • 敵（敵の巣含む）、炎は最低ひとつ配置してください。
  • 導火線は炎の真下からある程度の長さ(ジェームスが誘導出来る長さ)を延ばして配置してください。
  • 自作ステージは一つのファイルに最大５０ステージまで作れますが、少なくても問題有りません。総ステージ数が５０以下の場合はステージ番号１から順番にデータを保存して下さい。
