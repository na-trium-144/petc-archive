遊び方

+  遊び方
+  タイトル

STARTボタンを押すと始まります。
初めての方はオープニングが始まります

+  食べ物

食べ物では100Gで空腹度を回復出来ます。
お腹がすくとお腹空いたよーとキャラクターが言います。

+  じゃんけん

遊ぶではじゃんけんを遊べます。

+  散歩

散歩では海、森林、草原、洞窟、火山へ行けます。
そこでお金を拾ったりします。たまに落としたりもします。

+  カジノ

カジノでは丁半当てゲームとルーレットゲームが出来ます。ともに500Gです。

+  筋トレ

筋トレでは1000GでHPがアップすることがあります。
失敗もします。HPが一定を超えるとLVが上がります。
LVが一定を超えると進化します。

+  ステータス

ステータスでは現在のステータスが見れます。

+  銀行

銀行ではお金を預けたり引き出したり出来ます。

+  設定

設定では初期化とタイトルへとセーブして続けるが出来ます。

+  お出かけ

散歩で行けるところへお出かけ出来ます。

+  館

正式名称戻りの館。
ニワトリからヒヨコに戻れます。
元には戻せません。~GORD以外初期値に戻ります。

提案1
キャラを動かして宝箱を取ってGORDゲット。
ハズレはGORDを落とす
提案2
3つの分かれ道選んで当たりはGORDゲット。
ハズレはGORDが無いか落とす。
