2016年11月19日投稿 / 投稿者 : sou51さん / タグ : 投稿プログラム ゲーム シューティングゲーム

概要

プチコンマガジン創刊号に収録されていたシューティングゲーム「ソリッドガンナー」を 大幅に改造・パワーアップさせたシューティングゲームです。

必須環境

公式公開キーとして公開されている「ソリッドガンナー」の各種データが必要です。 オリジナル版ソリッドガンナーは公開キーJD2EE2QVよりダウンロードしてください。

遊び方

タイトル画面で、上下でモードを選択し、いずれかのボタンを押すと決定です。

GAME START

メインモードです。6セクション×2周の全12セクションクリアを目指します。

TRAINING

トレーニングモードです。メインモードで到達したセクションの中から好きなセクションを選び、様々な設定をして練習プレイができます。

KEY CONFIG

ゲーム中に使用するボタン設定ができます。

操作方法

メインモード操作方法

  • ショット(デフォルトB)
    • ショットを発射します。基本的に押しっぱなしになると思います。
  • ボム(デフォルトA)
    • ボムを使用します。スタイルセレクトでパワースタイルを選択した場合は、ブーストのON/OFFを切り替えます。
  • ハイパー(デフォルトY)
    • ハイパーモードを発動します。ハイパーゲージが満タンならそれを消費、オートボムOFFの場合はボムストックを1つ消費して使用できます。

トレーニングモード 設定画面 操作方法

  • 左右　項目の値の変更
  • 上下　項目の変更
  • L/R　Lで項目の最小値、Rで最大値に変更
  • STARTに合わせてA　設定した内容でゲーム開始
  • EXITに合わせてAもしくはB　タイトルに戻る
  • START・EXIT以外の項目でA　STARTに移動
  • START・EXIT以外の項目でB　EXITに移動

トレーニングモード プレイ中 操作方法

メインモードの操作に加え以下の機能があります。

  • リセット(デフォルトX)
    • トレーニングモードでプレイ中にすぐに中断して設定画面に戻ります。

キーコンフィグ

  • 変更したい項目を上下で選んで、ABXYLRのボタンを押すと設定を変えることができます。
  • USE OTHER 3DSを選択するとワイヤレス通信が始まります。(3DSのみ)
    • 親機になった3DSを画面表示、子機になった3DSをコントローラとしてプレイできる機能です。
    • ただし保存画面などのシステムダイアログは親機でしか操作できません。
    • 子機は通信が切れるまでの間上画面に「CONTROLLER MODE」と表示されます。
    • プチコンBIGでは選択できません

その他の操作

  • 十字キーの上とLとRとBを押しながら起動すると(厳密には偽デバイスチェック画面が終わるまでに押しっぱなしにすると)、ハイスコアのリセットができます。(オリジナル版からある操作)
  • Lを押しながら起動すると偽ブート画面と警告画面をすっ飛ばしていきなりタイトル画面から始まります。

攻略ポイントなど

ハイパーを使用すると敵の攻撃が激しくなるので(激しさはセクションクリアごとにリセット)、
クリア重視の場合はハイパーをあまり使わない方が先へ進みやすいかもしれない

ボムスタイル・ストロングスタイルでは、ボム数が最大数のときにボムアイテムを取得すると最大数が拡張されるので、
特にオートボムの場合、残機によってはミスしてからすぐに取得した方がお得。
(例えば、残機4、ボム数0/最大数3の場合、ボム取得しても3×3+1=10回から3×3+2=11回に増えるだけだが、
ミスしてから取得すると、残機3、ボム数4/最大数4となり、直前のミスを含めれば3×4+1=13回オートボムが発動されることになる)

ハイパーゲージが満タンのときは加算される蓄積点やヒット数に500HITから倍率がかかるので、
セクション序盤はHIT数を伸ばし、後半はゲージ満タンでプレイした方が点が伸びるはず
