プログラムリストあり

動画中でプログラムリストが紹介されている作品です。

2011/04/03 : 一画面ブロック崩し

  • 2 ちゃんねる プチコンスレ その3 >>647 より。1 画面プログラムによるブロック崩し。

2011/04/03 : Petit Scroll (プチコンゆっくりプログラム Vol.1)

  • 一画面プログラム。グラディウスの高速面を思い起こさせる壁よけゲーム。

2011/04/07 : 孤高の戦場

  • 2ちゃんねる プチコンスレ その3 >>679 より
  • 固定画面のシューティングゲーム。被弾せずに連続で敵戦車を倒していくと高得点のチャンス。プログラムは約600行。

2011/04/13 : convee

  • パレットアニメーションを使ったゲーム。
  • 浮遊する玉を動かして白い箱を避けていく。白い箱は飛び回っており、方向に応じて青と紫の軌跡を残す。玉がこの軌跡の上に乗ると左右に流されてしまう。

2011/04/15 : 地底最大の作戦

  • 2ちゃんねる プチコンスレ その3 >>802 より
  • I/O 1980 年 7 月号掲載の MZ-80K/C 用の名作ゲーム。プログラムは約 230 行。
  • プログラムリストは、3 つ目の「プログラムリスト完全版」の動画を参照してください。1 つ目の動画は、一部行に抜けがあります。

2011/04/17 : 疑似 MML プログラム

  • プチコンで MML による音楽を再生するプログラム。かなり短くまとまっている。

2011/04/21 : サブマリン

  • テキストベースによる懐かしい雰囲気の潜水艦ゲーム。
  • プログラムリストはこちら。

2011/04/23 : Petit Racing (プチコンゆっくりプログラム Vol.2)

  • 一画面プログラム。レースゲーム。

2011/04/24 : SPIDER

  • 複数の点からの長さが最短になるポイントを探すゲーム。

2011/04/28 : スーパーモンスター

  • MZ-80 用で人気があったI/O 掲載ゲームの移植。四方向から敵を押しつぶして全滅させる。QRコードはこちら

2011/04/28 : 晴れときどき爆弾

  • 昔の MSX-FAN に掲載されていたゲームの移植。
  • 空から降っていく爆弾をひたすら避けていくゲーム。

プログラムリストなし

動画中でプログラムリストが紹介されていない作品です。

2011/04/03 : マインスイーパー

  • おなじみのマインスイーパーです。
  • この Wiki の投稿プログラムコーナーには、リストが公開された別のマインスイーパーもあります。

2011/04/03 : GRPSHOP

  • タッチペンを活かしたお絵かきツール。十字キーでドット打ちも可能。

2011/04/03 : 一画面パネルめくり STG

  • 敵を倒した周辺のパネルがめくれているシューティングゲーム。めくれた下からは一枚絵が登場、上で紹介した GRPSHOP が活かされています。

2011/04/09 : すとらくちゃ

  • 道を配置し、棒人間を落としたり爆弾にぶつかったりしないように出口まで導くゲーム。

2011/04/09 : 激走ラビリンスを移植してみた

  • 1990 年に MSX・FAN に掲載された米屋のチャチャチャ氏作品「激走ラビリンス」を移植。米屋のチャチャチャ氏は当時 MSX ユーザーだった人たちの間で超有名だった凄腕プログラマー。
  • 原作はマシン語バリバリだったが、プチコン版はもちろんオール BASIC。
  • 2 つ目の動画は原作の MSX 版です。

2011/04/13 : 時計と背景

  • 背景と時計を別のプログラムで処理するテスト。詳しくは動画アップロード主のコメントを参照。

2011/04/16 : REFLECT EDGE

  • かなり本格的なシューティングゲーム。多彩な敵の攻撃や巨大ボスなど非常に凝った造り。
  • プログラムリストは 2500 行にものぼるとのこと。

  • ちなみに、プチコンから Windows に移植した版もある。

2011/04/17 : ICBM ジェネレーション

  • いわゆる酔っ払いゲーム。ただし、ステージの仕掛けが非常に豊富で飽きさせない。
  • どうやら、作者の方が昔 X68000 で作成したゲームのリメイクのようです。(4つ目の動画)

2011/04/17 : DSiWARE のプチコンで、GRADIUS もどきを作ってみている

  • 某横シューティングっぽいやつです。

2011/04/18 : プチコンでテトリス風ゲーム

  • テトリスです。ブロックの色や回転法則は TGM 風。

2011/04/18 : 晴れ時々コンテナ

  • ちょっと変わった落ちものパズルゲーム。フィールドを左右にずらして、コンテナを隙間無く積んでいくゲーム。

2011/04/18 : スプライトを使った縦 STG のテスト

  • スコアアタック的なシューティングゲーム。

2011/04/22 : ラリー X

  • あのラリー X です。驚異の再現性。

2011/04/23 : スターあれジャー

  • オーソドックスなシューティングゲーム。自機のパワーアップが豊富。
  • ハドソンのあのゲームの移植…ではない。むしろ初期ハイスコアはコナミ点という突っ込み待ち状態。

2011/04/24 : プチコンでマイコン

  • 現代によみがえる 4 ビットマイコンのシミュレータ。

2011/04/24 : 試作触手シューティングゲーム

  • 触手がうねうね動く自機の試作。

2011/04/25 : 妄想 CPU マイコン

  • 架空の CPU を搭載したマイコンシミュレータ。4 ビットマイコンシミュレータに影響を受けて制作。

2011/04/27 : PetitShaddai

  • あのエルシャダイがついに DSi でゲーム化で大丈夫か? 大丈夫だ、問題ない。

2011/04/28 : ピアノロール 《ドラム》

  • ピアノロールというか、リズムマシーン。ただしサンプルのものとは違い、ベロシティ (音の強弱) を個々のノートに設定可能。

2011/04/28 : プチコンでタイムボカン

  • タツノコ世代ではないので、元ネタが分かりませんでした…。

2011/04/30 : テトリスを作ってみる。

  • またまたテトリス。こちらはワールドルールに準拠で、テトリス DS っぽい感じ。

その他

プチコンのプログラムリストではない何か。

2011/04/22 : プチコン自動タイピングロボット「打ち込み君」

  • プチコンのプログラムを自動で打ち込んでくれるロボット。なんという謎の技術。

2011/04/25 : DSi で BASIC

  • なんとニンテンドー DSi ブラウザーで動作する BASIC。プチコン登場の 2 年前に制作されたとのこと。
