注意

スマイルブームさん、こあ/co.aさん、ありがとうございます。

アプリのインストールの方法

+  ...

スルメさんも説明を書いてくださりました。(こちらです)
スルメさん、ありがとうございます。

  1. 実行モードで「ＡＰＰＥＮＤ　”アプリのファイル名”(Enter)」と入力してください。
  2. 92行目のＤＡＴＡ文の値を1つ増やしてください。
  3. 93行目にＤＡＴＡ　”アプリの名前”，”アプリのラベル名”を追加してください。
    • アプリの名前はどんな名前を入力しても大丈夫です。
    • アプリのラベル名は＠以降の部分だけを入力してください。
      • 【例】＠ＴＥＳＴ→ＴＥＳＴ

「Noodel Software」の最新情報

開発しているもの

開発中止になるものがあるかも知れません。
「プチコン3号 SMILEBASIC」までには完成させたいです。

  • 「Noodel OS Version 2」を開発中。
    1から作り直したので「Noodel OS 2」などの名称で新たに投稿します。
    • 今は「Noodel OS 2(仮)」って読んでください。
  • 「Noodel OS DX」を開発中。
