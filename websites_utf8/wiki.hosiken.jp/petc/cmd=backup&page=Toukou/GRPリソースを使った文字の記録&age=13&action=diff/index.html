* プログラムリスト [#proglam]
**Ver 1.00 [#Ver100]
***プログラムリストの見方 [#proglaml]
灰色で囲んであるのは削除、青で囲んであるのは追加です。
#basic{{
GCLS 0:CLEAR:DIM A$(1529)
FOR I=0 TO 1528
LINPUT A$(I)
IF A$(I)=="" THEN END
FOR U=0 TO LEN(A$(I))
B$=MID$(A$(I),U,1)+A$
IF B$==A$ THEN U=LEN(A$(I)<COLOR>(blue){)}:NEXT:NEXT
GPAGE 1
GPSET OLDX,OLDY,ASC(B$)
OLDX=OLDX+1
IF OLDX>255 THEN OLDX=0:<COLOR>(#cccccc){?"PPP":}OLDY=OLDY+1
NEXT:NEXT
}}
**読み込み用 [#q498e761]
#basic{{
FOR Y=0 TO 191
FOR X=0 TO 255
PRINT CHR$(GSPOIT(X,Y));
IF LG==0 AND LG==G THEN END
LG=G
G=GSPOIT(X,Y)
NEXT:NEXT
}}
***このプログラムを実行する際の注意 [#i96eadf4]
Ver 1.2で実行してください。
エラーがでる恐れがあります。
