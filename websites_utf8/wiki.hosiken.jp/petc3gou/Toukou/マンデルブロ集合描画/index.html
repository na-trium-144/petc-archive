2014年11月30日投稿 / 投稿者 : hanzo / タグ : 投稿プログラム ツール 鑑賞プログラム

概要

3DSの画面を複素平面に見立てて、マンデルブロ集合を描画します。マンデルブロ集合については、ウィキペディア等をご参照ください。

  • プチコン3号の処理速度を概略的に確認したかったのと、3号で追加された"DEF"を使ってみたかったがために作成したプログラムです。
  • それにしても、プチコン3号は速いですね。学生時代に、同様のことをMSX BASICで試したことがありましたが、1画面描画するのにまる1日以上掛かった記憶が・・・（尤も、当時の作者のプログラムが未熟だったと言う事情もありますが・・・）

起動方法

ダウンロードしたプロジェクトから HNZMBRT を実行してください。

おしらせ
  • ダウンロードしたプロジェクトの中には、他にもいくつかの実行可能ファイルがあります。それらについては、以下リンクをご参照ください。
    • HNZTL（交通管制シミュレーター）
    • HNZHS（ビリヤード風ゲーム）
    • HNZMJG（麻雀役形成ゲーム）
    • HNZHC（ハミングをMMLに変換）
    • HNZVT（絶対音感ボイストレーニング）
    • HNZCDRV（崖っぷちハンドルドライブ3D）
    • HNZMBRT（マンデルブロ集合描画）
    • HNZLND（着陸ゲーム）
  • もし、同名のhanzo soft製のプロジェクトを既にお持ちの場合は、お手数ですが、既存のプロジェクトを消去してから、本プロジェクトをダウンロード願います。

画面紹介

操作方法

下画面を使って操作します。

  • LOOP COUNT
    • 複素漸化式の収束判定のための最大繰り返し回数を設定します。この値が大きいほど、描画はきめ細かくなりますが、描画速度は著しく遅くなります。
  • HUE / SAT / VAL / GAMMA
    • 表示の色に関するパラメータを変更します。
  • INIT PARAMETERS
    • LOOP COUNT / HUE / SAT / VAL / GAMMA を初期値に戻します。
  • Zoom
    • 拡大部分選択画面に移ります。拡大部分選択画面では、タッチペンを使って拡大したい部分を選択します。
  • INIT COORDINATES
    • 初期座標に戻して再描画します。
おしらせ
  • LOOP COUNT / HUE / SAT / VAL / GAMMA および、拡大部分選択操作は、タッチペンを離してから約2秒後に発動します。2秒以内に再操作を行った場合、先の操作の発動はキャンセルされます。
  • 拡大部分を選択するときは、「黒い部分」と「黒くない部分」の境界部分を選択すると、複雑で美麗な描画を楽しめます（下図参照）。

公開キー

プチコン3号用の公開キーです。トップメニューの「作品公開とダウンロード」から公開キーを入力すると、作品をダウンロードすることができます。

プチコン3号/BIG 公開キー
1DVK34J
