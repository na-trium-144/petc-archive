//-----------------------------------------------------------------------------------------------
// おことわり : プチコン4は家庭用ゲーム機向けのソフトですので、
// 作成基準的に絶対に実装できない機能や、セキュリティ上の理由でなくなった機能があります。
// (例) ・データセーブ時に確認ダイアログを表示しない

衝撃のプチコン登場から 8 年。ハードもソフト世代交代が進み、驚きの進化を遂げました。もはやプチコン4 に不可能はない?

それでも人間の欲望は止まりません。プチコン4 で、こんなこといいな できたらいいなと思うネタがあったら、ここに書き連ねていってください。

* 要望一覧 [#z4b64433]
** 命令 [#ebd7bf89]
- Toy-Conのドライブキット・VRキットの対応
-- Ver4.2.0でドライブキットは実装された

- &basic(){TALK};命令の強化(mkIIと同じように)
-- 音声は変更されたが強化はされてない(機械音感が増した)

- ローカル通信の復活(&basic(){MP};~~形式の命令)
-- 3号にあったが、何故か消された
-- 画面分割と複数のコントローラーを使用すれば一応再現出来ると思う

- Switchが何のモードか、Switch Liteかどうかが分かるように
-- &basic(){XCTRLSTYLE};命令などを使えばTV・テーブル・携帯モードは反映される

- 定数を他のスロットで使えるように
--&basic(){COMMON CONST};のように
--もしくは&basic(){VAR};のように&basic(){CONST(”0:#HOGE”)};という形を使えるように

** 3号との互換性 [#wb2fbf08]
- 3号との公開キーの互換性(4の公開キーを3号でも使用可能に)
-- 3号のプログラムは、公開キーをインターネット経由で入力すれば4でもダウンロードできる(使うためにはプログラムを4用に編集・変換する必要あり)

- プログラムを3号用・4用に変換(3号のプログラム~4用・4のプログラム~3号用にそれぞれ変換して送れるように)
-- 今のところ自分で編集するか、有志の方が作成・配布している変換プログラムを利用するしかない

** その他 [#k542694c]
- ｢セーブ領域の拡張｣の限界容量を今より多く(できれば8GBまで)

- セーブ領域の残り容量を確認できるように

- 外部からのデータ転送システム

- HD振動の強さ、振動する時間などを細かく決められるように(何種類かサンプルがあるが)

- ポケモンタイピングのキーボードを使えるように(SwitchはBluetoothに対応している)

- Nintendo Switch Onlineに加入していればサーバー利用券不要でアップロードできるように

-MAIN.PRG以外をプロジェクトを押した際に実行するファイルにできるように
