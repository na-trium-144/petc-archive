プチコンコミュニティの避難先一覧

Lobi プチコン3号 (仮設ミバ難民キャンプ)

  • URL : https://web.lobi.co/group/31b5d2b497e0154dd0034a016bc8cbd6bbe00011
  • 投稿は要ユーザー登録
  • 専用アプリからアクセス可能
  • 3DS からのアクセスは不可能 (容量オーバーになる)

Pixiv Mohverse (渋支部)

  • URL : https://www.pixiv.net/group/?id=37977
  • プチコンのお絵かきツールと、描いた作品を共有するコミュニティ
  • 全年齢対象
  • 関連する質問も可能
  • 投稿にはPixivアカウントが必要

Petitverse(プチバース) プチコンコミュニティ

  • URL : http://petitverse.hosiken.jp/community/petitcom/
  • Miiverse によく似た見た目の掲示板で、ユーザー登録不要で従前通りプレイ日記とトピックが書き込めます。ただしなりすまし防止のため投稿の際に投稿用パスワードを入力する必要があり、忘れると以前の名前が使えなくなる。
  • プチコンコミュニティ専用です (それ以外のコミュニティはありません)
  • 3DS からアクセス可能
  • プチコン3号・BIGまとめWiki(当Wiki)管理人が運営

Twitter

  • URL(公式ハッシュタグ #petitcom の検索結果) : https://twitter.com/hashtag/petitcom?f=tweets
  • URL(作品公開キー用タグ #sbkey の検索結果) : https://twitter.com/hashtag/sbkey?f=tweets
  • その他プチコン関連単語、関連ハッシュタグを含めたの検索結果
  • 投稿は要ユーザー登録 日本ではとても普及しているので既にアカウントを持っているのならここが良いだろう。
  • Miiverseでの通称は「青鳥」
  • 3DS から一応アクセス可能 newの場合はフィルタリング解除か、Google翻訳等で回避する必要がある。

BlueHood プチコンコミュニティ

  • URL : http://bluehood.net/view?comm_id=23be6f2b
  • Miiverse の感覚でTwitterの投稿・閲覧ができる
  • 手書き投稿や写真添付に対応。
  • 投稿にはTwitterアカウントが必要
  • 3DS対応

プチコン公式サイト ユーザー作品紹介

  • URL : http://smilebasic.com/user/
  • 交流要素はありませんが、オフィシャルな作品公開の場です
  • フォームから申請すれば作品を投稿可能
  • 3DS からギリギリアクセス可能
  • ユーザ作品の投稿は暫く休止するそうです。

ミナコイチャット

  • URL(部屋検索結果) : http://www.3751chat.com/?tag_id=93909#freerooms
  • ユーザー登録不要
  • Miiverseでの通称は「3751」
  • 掲示板ではなく、チャットです
  • 3DS からアクセス可能

ミバちゃんねる プチコンスレッド

  • URL : https://zawazawa.jp/urauramiba/topic/48
  • 無料掲示板サービス zawazawa に作られたプチコンスレッド
  • zawazawa はユーザー登録して書き込める掲示板

Nextverse プチコンコミュニティ

  • URL : http://seatex.webcrow.jp/nextverse/index.php?titleid=11720
  • Miiverse の感覚でTwitterの投稿・閲覧ができる
  • 投稿にはTwitterアカウントが必要
  • 投稿は少なめ (全体の投稿はトップページを参照

miniverse プチコンシリーズ

  • URL : http://minimal97.mydns.jp:8123/frm/viewforum.php?f=79
  • 名前に日本語は使えず、FAQに英語の箇所が多い。
