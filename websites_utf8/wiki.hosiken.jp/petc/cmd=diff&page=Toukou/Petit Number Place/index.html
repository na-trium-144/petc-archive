* 概要 [#Summary]
よくあるナンプレです。~
問題を解いたり作ったりする機能はありませんが、250問を編集・保存できます。~
人気パズルなだけあって、すでにプチコンによる作品がいくつもあり、また今現在も新作を作っている人はたくさんいますが、自分の好みで、自分が納得がいく、自分が欲しいものを作ってみました。~
まだ追加したい機能はいろいろありますが、とりあえず人目に出しても問題ない程度にまで仕上げたので公開します。

世界的に知られているゲームなのでメニューの文字などに（いい加減な）英語を使っていますが、簡単だと思います。~
おもにタッチペンで操作しますが、補助的にボタンを使えるシーンもあります。

問題の編集は、自分で設定したレベル（自動判定はできません）、目標時間（49時間59分59秒まで）、コメント（32文字まで）を設定できます。~

問題を解く時は、通常の数字の入力、候補数字の入力、数字を強調表示するフィルター、色分け、ペン書きができます。~
また、数字、候補数字は、640手まで履歴をさかのぼれます。

候補数字は、表示なし・４つまで表示・９つすべて表示・数によって切り替えるAUTOモードをその場で簡単に切り替えられます。

強調表示のフィルター機能は、数字の上で2回タッチするか、問題の数字の上にカーソルを置いた状態で入力パネルの数字を押すと、その数字が強調表示されます。~
また、問題の数字や、入力した数字の上にカーソルを置いた状態で入力パネルの候補数字をタッチすると、候補数字を強調表示します。~
問題の数字の上で入力パネルの削除を押すと、候補数字が２つ残っているマスが強調表示されます。

色分けは、通常の数字と、候補数字を別々に設定できます（どちらを色分けするかは、チェックボックスで切替できます）。~
［Ｒ］アイコンのタッチで全部消えてしまうので注意してください（アイコンの外にタッチをずらしてから離せば消えません）。

ペン書きは、赤と青の細いペンを用意しています。消しゴムで消せます。~
［Ｒ］アイコンのタッチで全部消えてしまうので注意してください（アイコンの外にタッチをずらしてから離せば消えません）。

解いている途中の問題は、３つまで保存できます。~
入力数字の履歴と、最後の状態での色分けも保存されています。~
ペン書き内容は保存されません。

プレイ途中に保存した問題をプレイ再開した場合は、RETRYメニューの中にREWINDボタンが追加されます。~
REWINDを押すと、前回保存したところまで戻ります。~
ただし、プレイ時間は戻りません。~
プレイ途中に保存した問題は、クリアした時、リトライした時、ギブアップした時に自動的に削除されます。


** 遊び方 [#Komidashi1]
メニュー画面から、適当に選択して遊んでみてください。

各問題を示すナンバーの枠の中で、左上に表示される小さい四角は、自分で設定したレベルです。~
右上は、何らかのデータが存在する時は「○」（コメントにスペースが入っているだけでも表示）、クリアしている場合は「◎」、目標時間を設定していて、それより短時間でクリアしている場合は「♪」のマークが表示されます。
右下に時計マークがついている場合は、プレイ中に保存されたもので、この画面から「CONT」ですぐ実行、または「G-UP」でギブアップできます。~

右下のシステムアイコンの機能の中にペーストと削除（ゴミ箱アイコン）がありますが、解いている途中で保存されている問題への上書きペーストや削除はできないようになっています。~
また、削除は、削除と同時にコピーも行っているので、ペーストすれば復活できます。

* QR コード [#QR]

// QR コード化すると、枚数が半分ぐらいに減ります。ぜひお試し下さい。
// 【重要】添付できるのは画像ファイルのみです。zip ファイルは解凍してください。

// ページを作成すると、QR コードの添付ができるようになります。(作成する前はできません)
// 作成後のページの右上、ウィキメニューにある「添付」から画像をアップロードできます。
// (添付ファイルはページごとに別個に管理されています。必ず作成後のページに添付してください)
// QR コードをアップロードしたら、以下の書式でページ内に貼り付けられます。
// #ref(アップロードしたファイル名)

// QR コードを大きくしたい場合は、倍率指定ができます。コンマは必ず 2 つ指定してください。
// ブラウザによっては補完がかかってボケますが、それでも問題なく読み込み可能です。
