概要

ある日お団子が食べたくなった狐少女の桔音 紺は、
お団子屋さんへ行くことにしました。が・・・？

ちょこっとRPGです

※この作品は第一回非公式プチコンコンテスト出展作品です

説明

マップ移動時

十字キー・・・移動
Aボタン・・・調べる

メッセージウィンドウ表示時

Aボタン、下ボタン・・・メッセージを読む
Bボタン・・・早送り

バグ情報

現在以下の不具合が確認されています

誤植

修正箇所は
866行目 MES$(0)="そうだよ、けっきくは"　　　→　　　MES$(0)="そうだよ、けっきょくは"
です。(ただの誤植なので気にならなければ、直さなくても問題ありません)

ゲームオーバーやクリアでタイトルに戻った後、一部の変数が初期化されない

修正箇所は、
1042行目 IF !F THEN @TITLE　　　→　　　IF !F THEN GOSUB @START:GOTO @TITLE
1047行目 GOTO @TITLE　　　→　　　GOSUB @START:GOTO @TITLE
1316行目 GOTO @TITLE　　　→　　　GOSUB @START:GOTO @TITLE
です。(直さなくても、タイトルに戻った後一度プログラムを終了すれば問題ありません)

ハイスコアの一部が表示されない

修正箇所は、
1506行目
IF MID$(MEM$,25,1)=="3" THEN PRINT "えんでぃんぐA"
　　　　　　　　　　　↓
IF MID$(MEM$,25,1)=="0" THEN PRINT "えんでぃんぐA"

1507行目
IF MID$(MEM$,25,1)=="4" THEN PRINT "えんでぃんぐB"
　　　　　　　　　　　↓
IF MID$(MEM$,25,1)=="1" THEN PRINT "えんでぃんぐB"
です。(直さなくても、ゲームの進行に問題はありません)
