便利ツール

プログラム実行
  • PTC-Interpreter(未完成、開発中止) - PTCファイルを実行するインタプリタ。Javaで実行できる。
  • PTC-EmkII(開発中、未完成) - PTC-Interpreterの後継。Linux用(ダウンロードして自分でビルドする必要がある)。Windowsでも一応動かせる。
QRコード作成
  • PetitEditor - TXT->QR。ブラウザ上で動作。
  • PTC2QR - PTC->QR。ダウンロードウェア。Windows専用。
  • PTCUtilities - PTC,TXT->QR。ダウンロードウェア。Windows,Mac専用。
  • CHRED on Android - PTC->QR。Android端末専用。 ※公開終了
プログラムリスト変換
  • PTC→TXT 変換 - PTC->TXT。ブラウザ上で動作。Firefox,Chrome,Opera専用。JavaScriptのFile APIを使用。
  • SDメモリカードに書込んだプチコンのプログラムを読込む - PTC->TXT。ダウンロードウェア。Windows専用。
  • PetitConverter - PTC->TXT。Ruby製。ダウンロードウェア。ダウンロードキーワードは「ptc」。Windows専用。
  • PTCUtilities - PTC,QR->TXT。ダウンロードウェア。Windows,Mac専用。
  • プログラムのインポート - プチコンのプログラムを圧縮して、展開プログラム+DATA文に変換します。
  • GRPからプログラム書き出し - プロクラムのインポートでGRPに保存したプログラムを、KEY経由でPRGリソースに書き出します。
  • CHRED on Android - PTC,QR->TXT。Android端末専用。 ※公開終了
PTC ファイル作成
  • TXT→PTC変換 - TXT->PTC。ブラウザ上で動作。
  • パッケージ形式変換 - 複数PTC->パッケージ形式PTC。ブラウザ上で動作。Firefox,Chrome,Opera専用。JavaScriptのFile APIを使用。
  • PetitConverter - TXT->PTC、もしくはPTC->TXT。Ruby製。ダウンロードウェア。ダウンロードキーワードは「ptc」。Windows向け。
  • PTCUtilities - TXT,QR->PTC。ダウンロードウェア。Windows,Mac専用。
  • CHRED on Android - CHR,COL編集ツール。QR->PTCもできる。Android端末専用。 ※公開終了
プログラムリスト作成支援
  • プチコンプログラム短縮 - プログラムを入力すると、省けるところを徹底的に省いたものにしてくれる。1画面プログラムを極めたい方などへ
  • プチコン文法チェッカー - プログラムを入力すると、文法のエラーチェックができる
  • スマホ用命令表 - iPhoneなどで表示しやすいように変換された命令表
  • vim用のシンタックスファイル - vim でプチコンのソースをシンタックスハイライトさせるための設定ファイル。
  • vim用のインデントファイル - vim でプチコンのソースのインデントを設定するファイル。
GRP / グラフィック関連
  • 画像変換ツール - BMP ファイルを、DATA 文や PTCファイルに変換してくれる Windows 用ツール
  • SDカードGRPビューワ - SDカード内にあるGRP画像を１枚ずつ表示したりＰＮＧ画像として保存できる。
  • GIF表示 - GIFファイルをプチコンで表示するプログラムを生成します。Firefox / Chrome 専用。
  • GIF書き出し - GRPリソースをGIFに変換してQRコード化します。
  • Petitcom GRP Exporter - 画像ファイル -> GRP,COL QRコード生成。ブラウザ上で動作。
  • BMPファイル変換 - TINY野郎さんが個人作成・使用しているツール。ブラウザ上で動作。
Web 公開支援
  • PTC2HTML - PTC->プチコンフォントのHTML。ダウンロードウェア。Windows専用。
  • PTCUtilities - TXT,QR,PTC->QRコードのHTML。ダウンロードウェア。Windows,Mac専用。
