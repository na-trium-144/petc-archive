ゲーム内容

ニュー速VIPのキャラクターが競う全6コースのレースゲームです。
?マークのパネルを踏むとアイテムをゲットできます。
コース上の星を集めると速度が上がります。
5週するとゴールです。

Wiiリモコンが2つあれば2人対戦プレイができます。

操作方法

  • 3DS/ゲームパッド
    • 十字キー左右 : 旋回
    • Aボタン : 加速
    • Bボタン : ブレーキ
    • X,Lボタン : アイテム使用
    • Rボタン : ポーズ
  • Wiiリモコン
    • 十字キー左右 : 旋回
    • 2ボタン : 加速
    • 1ボタン : ブレーキ
    • Bボタン : アイテム使用
    • +ボタン : ポーズ

記録を見る画面でL+R+Xを押すと、記録を消去できます。

アイテム

緑の玉
真っ直ぐに発射され、当たったカートをスピンさせます。
壁にぶつかると跳ね返ります。
赤い玉
1つ前のカートを追跡します。
エビフライ
コース上に設置され、当たったカートをスピンさせます。
加速缶
一瞬速度が上がります。
無敵缶
一定時間無敵になり、当たったカートをスピンさせます。
また、草地に入っても速度が落ちません。
隕石
自分以外のカートを一定時間スピードダウンさせます。

攻略のコツ
+  攻略のコツ
  1. 全キャラ共通
    1. なるべく早い段階で星を15枚集めよう！
    2. 加速台を確実に踏もう！
    3. ライバルの前や後ろは、攻撃を受けやすいので近づくな！
    4. コンピュータは同じラインを通るので、そのラインにエビフライを置くと当たりやすいぞ！
    5. 攻撃系アイテムを使う時は、ライバルが無敵でないか確認しよう！
    6. 無敵状態なら最高速は上がるが加速は上がらないので、再スタート時には無敵缶を使うな！
  2. 加速が低いキャラ
    • 加速缶が出たら、攻撃を受けた時の保険にとっておこう！
  3. 最高速が低いキャラ
    • 加速の低いライバルをとことん攻撃しよう！
