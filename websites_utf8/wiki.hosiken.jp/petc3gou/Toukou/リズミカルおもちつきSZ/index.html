最終ページ更新 12/23 >>> SZ5.0 を公開しました！

これはなんですか？

☆TAKA☆氏制作の神作であるリズミカルおもちつき( リズミカルおもちつきSP ) をわたし すずちゃん が勝手に改造したプログラムです。

いわばn番煎じの派生作品です。

ちなみに某Taikoに近づけるつもりはまったくありません。

おもちSPとの違いはなんですか？

  • トロフィーシステムからランクシステムに変わりました
  • クリア条件が精度ではなくHP制になりました
  • 新しいゲームモードが追加されました
  • 譜面オプションがいくつか増えました
  • スコアシステム・ppシステムが導入されました
  • 精度システムが改変されました
  • 譜面にセクション(区間)が設定できます
  • なんか黄色いのがいますね。よろしくおねがいします。
  • ハイスコアがちゃんと記録されるようになったっぽいです。わかりません。

ほかにもちょこちょこ違うところがありますよ。

スコアシステム

コンボ数やオプションによってスコアが算出されます。

ppシステム

スコアシステムとは違い、譜面の難易度なども考慮して算出されます。

高いppがほしいならまずはフルコンボをすることです。

譜面オプション

いくつかの譜面オプションを追加しました。

また、既存の譜面オプションもすこしだけ変わっています、たぶん。

譜面オプションを設定することでスコアに倍率がかかります。

Arrow ModeとMania Modeではオート[AT]のみ利用できます。

一部譜面ではしなない[NF]とオート[AT]のみ利用できます。

ランクシステム

おもちつきSZではトロフィーシステムではなくランクシステムになりました。

ランク条件
XXオプション +HD 精度100% フルコンボ スコア倍率1.00以上
XSオプション +HD 精度95%以上 フルコンボ スコア倍率1.00以上
SS精度100% フルコンボ スコア倍率1.00以上
S精度95%以上 フルコンボ スコア倍率1.00以上
A精度90%以上
B精度80%以上
C精度70%以上
D精度70%未満

スコア倍率が1.00倍未満だとランクがAまでになります

セクションシステム

譜面にセクション(区間)が設定できるようになりました。

1セクション内の精度が80%以上ならPass、80未満ならFailと判定されます。

新しい音符が増えた

新しく黄色い音符が増えました。

これは ホールド(仮称)音符 で、赤音符ボタンまたは青音符ボタン長押しで処理します。

タッチでも同様ですよ、たぶん。

判定は広めです。簡単な譜面向けかも。

ちょっとバグがあります、がんばってください(?)

新モード Arrow Mode

従来の 赤音符 と 青音符 を叩くのではなく、赤音符 と 緑音符 を叩くモードです。

赤音符は A B X Y どのボタンでも叩くことができますが、

緑音符は 音符に対応した十字キー でないと叩くことができません。

おもちモードの譜面を変換して遊ぶことができます。

新モード Mania Mode

4レーンに降ってくる音符を叩くモードです。

左から ↑ → Y X ボタンが対応しています。

おもちモードの譜面を変換して遊ぶことができます。
