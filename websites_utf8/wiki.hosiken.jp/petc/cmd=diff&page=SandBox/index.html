*BASICプログラムの記述 [#mc3c1296]

 #basic{{
 PRINT "さようならさようなら!"
 }}

と書くと、BASICプログラムをプチコンフォントで記述できます。

#basic{{
PRINT "さようならさようなら!"
}}

 &basic{ RUN };

と書くと、&basic{ RUN }; こんな感じで文章中にも使えます。
