投稿プログラム : PETIT RUN mkII

2012年09月15日投稿 / 投稿者 : おちゃめ / タグ : 一般 ゲーム 1画面プログラム

概要

1画面でできたタイムアタック専用の疑似3Dレースゲームです。

説明

十字ボタンの左右で移動、Aボタンを押せば加速し離せば減速します。
急カーブはかなり減速しないと曲がれません。
ゴールしたらAボタンでリトライできます。

デフォのコースの自己ベストは20秒89です。

コースは簡単に造れるのでオリジナルのコースを造って友人同士でタイムを競ってください。
コース変更を行うには5行目のＣ＄の内容を変えればいいだけです。

コースの例
コース15567654345434576564312555デフォのコース
コース211991199119911991199テクニカルコース
コース355555559555555515555555高速コース

「1」が左急カーブ、「9」が右急カーブとなっており、数字によってカーブの度合いが変わります。「1」と「9」の中間である「5」は直線になります。

見どころ

独自アルゴリズムによって3D表示と視点移動を行っています。
自サイトで発表済みのPETIT RUNに最高レベルのリスト短縮を施して大幅な機能アップを行っています。
60fps、処理落ち無しで動作するのでタイムアタックが熱いです。

スクリーンショット・動画

※一発撮りで画面が少し斜めになっていますが気にしないでください。

QR コード

プチコンmkII 向けの QR コードです。

PRG:OCHAPRN2
1/22/2

プログラムリスト

リストの分解、再構築によって1画面で使える29文字x24行＝696文字のうちわずが空白部分は6文字となっています。
限界までリスト短縮を行った1画面プログラムを作ったことがある人ならばこのすごさが分かると思います。
