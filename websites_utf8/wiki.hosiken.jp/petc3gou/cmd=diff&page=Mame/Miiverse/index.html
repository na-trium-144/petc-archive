#title(Miiverse終了後の避難先まとめ)

Miiverse は 2017年11月8日にサービスを終了しました。

そして、プチコンコミュニティにいた人は今後どこに行けばいいのか? どこに散ってしまうのか? をまとめました。

#contents

* プチコンコミュニティの避難先一覧 [#petithinan]
** Lobi プチコン3号 (仮設ミバ難民キャンプ) [#lobipetit3]
- URL : https://web.lobi.co/group/31b5d2b497e0154dd0034a016bc8cbd6bbe00011
- 投稿は要ユーザー登録
- 専用アプリからアクセス可能
- 3DS からのアクセスは不可能 (容量オーバーになる)
- 過疎化している

** Pixiv Mohverse (渋支部) [#pixivsibusibu]
- URL : https://www.pixiv.net/group/?id=37977
- プチコンのお絵かきツールと、描いた作品を共有するコミュニティ
- 全年齢対象
- 関連する質問も可能
- 投稿にはPixivアカウントが必要

** Petitverse(プチバース) プチコンコミュニティ [#petitverse]
- URL : http://petitverse.hosiken.jp/community/petitcom/
- Miiverse によく似た見た目の掲示板で、ユーザー登録不要で従前通りプレイ日記とトピックが書き込めます。ただしなりすまし防止のため投稿の際に投稿用パスワードを入力する必要があり、忘れると以前の名前が使えなくなる。
- プチコンコミュニティ専用です (それ以外のコミュニティはありません)
- 3DS からアクセス可能
- プチコン3号・BIGまとめWiki(当Wiki)管理人が運営

** Twitter [#twitter]
- URL(公式ハッシュタグ #petitcom の検索結果) : https://twitter.com/hashtag/petitcom?f=tweets
- URL(作品公開キー用タグ #sbkey の検索結果) : https://twitter.com/hashtag/sbkey?f=tweets
- [[その他プチコン関連単語、関連ハッシュタグを含めたの検索結果:https://twitter.com/search?f=tweets&q=(%E3%83%97%E3%83%81%E3%82%B3%E3%83%B3%20OR%20%23petitcom%20OR%20%23sbkey%20OR%20%23smilebasic)%20-source%3Atwittbot.net%20-source%3AIFTTT%20-source%3Adlvr.it%20-miracrear%20-fm7mei%20-bot%20-%22%E8%85%95%E6%99%82%E8%A8%88%22%20-%E5%9C%A7%20-%22%E3%82%AF%E3%83%AC%E3%82%AA%E3%82%B9%22&src=typd]]
- 投稿は要ユーザー登録 日本ではとても普及しているので既にアカウントを持っているのならここが良いだろう。
- Miiverseでの通称は「青鳥」
- 3DS から一応アクセス可能 newの場合はフィルタリング解除か、Google翻訳等で回避する必要がある。
