* 概要 [#Summary]
-マインスイーパを自分なりに作ってみました。
-パッケージ型に挑戦しました。
-左利き対応です。
-完成まで3日以上かかりました…
-著作権大丈夫かな…

** 操作方法 [#Komidashi1]

*** タイトル画面 [#Komidashi1-1]
-↑,X  ：ハイスコアのセーブ
-タッチ：ゲームを開始します
-←,A：???
*** ゲーム中 [#Komidashi1-2]
-タッチ：パネルを開きます
-→,L,Y,R+タッチ
--パネルをタッチすると旗を立てます
--数字をタッチすると、その数字と周囲8マスの旗の数が同じなら残りを開きます
-↓,B+タッチ：？をつけます
-START:タイトルに戻ります

|~1/7|~2/7|~3/7|
|&ref(mineswp0.png);|&ref(mineswp1.png);|&ref(mineswp2.png);|
|~4/7|~5/7|~6/7|
|&ref(mineswp3.png);|&ref(mineswp4.png);|&ref(mineswp5.png);|
//|>|>|~7/7|
//|>|>|&ref(mineswp6.png);|
|~7/7|||
|&ref(mineswp6.png);|||

// 【重要】旧プチコンから mkII に転送したプログラムは、mkII で保存しなおしてから
// QR コード化すると、枚数が半分ぐらいに減ります。ぜひお試し下さい。
// 【重要】添付できるのは画像ファイルのみです。zip ファイルは解凍してください。

// 作成後のページの右上、ウィキメニューにある「添付」から画像をアップロードできます。
// (添付ファイルはページごとに別個に管理されています。必ず作成後のページに添付してください)
// QR コードをアップロードしたら、以下の書式でページ内に貼り付けられます。
// #ref(アップロードしたファイル名)

// QR コードを大きくしたい場合は、倍率指定ができます。コンマは必ず 2 つ指定してください。
// ブラウザによっては補完がかかってボケますが、それでも問題なく読み込み可能です。
// #ref(アップロードしたファイル名,,200%)

// 複数の QR コードを貼り付けるために表組みを用いる場合、&ref(); を使用します。

// |>|>|~PRG:FILENAME|
