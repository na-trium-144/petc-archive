2012年08月12日投稿 / 投稿者 : タマゴさん / タグ : 投稿プログラム ゲーム QRコード

概要

走ってくる車をよけるゲームです。 全16ステージあります。 それほど難しくありません。

気になる点が多いかも知れませんが、ご了承ください。

DATA文でステージの改造も可能です。 総数 2257 回

※Ver2.1に更新してから遊んでください。

操作方法

メインゲーム
  • 十字キー：移動
  • 十字キー+B：ダッシュ

車に当たらないで道路を渡りきれたら、ステージクリア。

エンドレス
  • 十字キー：移動
  • 十字キー+B：ダッシュ

車に当たらないで道路内に居て、ハイスコアをたたきだそう。

バトルゲーム

◯赤い髪のプレイヤー

  • 十字キー：移動
  • 十字キー+L：ダッシュ

◯青い髪のプレイヤー

  • ABXYキー：移動
  • ABXYキー+R：ダッシュ

どれ位道路内に居れるか競い合おう。

改造方法

DATA a,b,c,d,e,f,g,h,i,j

  • a:車がでる確率（１フレームごとa回に１回）
  • b:車の速さ
  • c:車の列の数（下りと上りで１列）
  • d:同じ方向に向かう車同士の間のX軸(１列の場合、この数字は関係ない）
  • e:一番左側の上りの車の列のX座標
  • f:一番左側の下りの車の列のX座標
  • g:制限時間
  • h:プレイヤーの初期状態のX座標
  • i:プレイヤーの初期状態のY座標
  • j:ゴールラインのX座標（プレイヤーのX座標＞ｊだったらステージクリア）
  • 一つ一つのBGの番号は、DATA文のASCコードです。

※このプログラムを再度セーブしたい場合は ＳＡＶＥ”ファイル名”，”Ｄ８００１７０１” と入力してください。
