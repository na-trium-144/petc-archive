2018年04月13日投稿 / 2021年12月05日更新 / 投稿者 : 天郷思音 / タグ : 投稿プログラム 実用ツール エディタ
検索用：【天郷思音】

概要

プリセット素材でマップを作る人におすすめの、マップ作成ツールです。
現在のバージョンは150です。

特徴

その1.パネルセクション

マップのデータと一緒に、マップチップの画像を分割して保存する機能です。
「ちょこっとだけ自分で描いた絵を使いたい！」時に便利です。
もちろん圧縮済み。GRPファイルを添付するよりもずっと軽量です。

その2.マクロペン

細長い建物だけでなく、複雑な組み合わせのチップを一度における機能です。
マクロペンの登録が面倒な方のために、簡易版の「建物ペン」もあります。（2*2マスで固定）

※マクロペンの情報はマップとは別に保存します。「OPTION」からセーブしてください。

操作方法

  • タッチペン：チップを置く / チップを選ぶ
  • スライドパッド：スクロール
  • →：押している間だけBIGペン（1度に4*4マスに同じチップを置くペン）
  • Y：3秒押しで編集中のレイヤーを消去します
  • ←　L　Rのいずれかでメニュー画面を開きます

メニュー画面

  • 家：建物ペンのオン / オフ
    • 建物ペンは2*2マスの絵が繋がるペンです。プリセット素材のお城やデパートに便利です。
  • なし / 90° / 180° / 270° / ←→ / ↑↓：アトリビュートの切り替え
  • SUPER COPY：スーパーコピー（全レイヤーコピー）機能
  • MAKE MACRO：マクロペンの登録
  • OPTION：その他の機能・設定

作ったマップの呼び出し方

手順
  1. RMBライブラリを使用可能にします。
  2. 必要なら、画面を初期化します。
  3. BG画面の面積を、必要に応じて拡張します。
  4. RMB_LOAD命令を呼んで、マップをロードします。
サンプルコード

前項の「手順」を実際にプログラムにしたものです。
そのままコピーしてお使いいただけます。
（"FILENAME"は実際のファイル名に書き換えてください。）

  1. ＥＸＥＣ　”ＰＲＧ３：ＲＭＢ＿ＬＩＢ．ＰＲＧ”
  2. ＡＣＬＳ
  3. ＦＯＲ　Ｉ＝０　ＴＯ　３
  4. 　ＢＧＳＣＲＥＥＮ　Ｉ，１２８，１２７
  5. ＮＥＸＴ
  6. ＲＭＢＬＯＡＤ　”ＦＩＬＥＮＡＭＥ”，０，３　ＯＵＴ　Ｉ

RMBLOADの書式は
RMBLOAD ファイル名,開始レイヤー,終了レイヤー OUT パネルセクション数
です。

  • ファイル名…拡張子は必ず省きます。
  • 開始・終了レイヤー…この範囲のレイヤーにマップチップが置かれます。開始≦終了でなければなりません。
    • 現在のバージョンでは、開始レイヤーは0のみ対応しています。
  • パネルセクション数…戻り値です。再度セーブする場合に使う値です。ロードのみの場合は値を捨てて構いません。

スクリーンショット・動画

※スクリーンショットは、プチコン本体のスクショ機能で撮った物に、劣化を軽減する加工をしています。

仕様上の制約

  • アトリビュートの併用はあえてできないように設計しています。
    • （※左右反転+上下反転=180°回転なのでうまく考えて使ってください。）
    • 透明チップ（0番）は強制で「アトリビュートなし」になります。
  • BGを重ねる順番は固定です。
    • 変更機能を検討しています。（上から順に3,2,1,0または0,1,2,3のどちらかが選べるように）

希望など

  • 作成したマップをゲームに組み込む時の不便点の把握
  • 機能追加
    • チップサイズ変更も規格上盛り込み準備中
    • 当たり判定の設定機能

RMB形式について

  • Run-length-encoding Multi-layer Bgの略です。
    • ランレングス圧縮された複数レイヤーのBGという意味です。
  • マップ4レイヤー+BG画像の一部を1つのファイルで保存できるフォーマットです。
  • デフォルトでランレングス圧縮が効いていることが特徴です。
  • RMB形式の仕様

公開キー

プチコン3号用の公開キーです。トップメニューの「作品公開とダウンロード」から公開キーを入力すると、作品をダウンロードすることができます。

ver.150　最新版
VP5QN3C4

プロジェクト内容

  • TXT:-EDITER.PRG…マップエディタ本体
  • TXT:MAP_VIEW.PRG…簡易ビューア
  • TXT:RMB_LIB.PRG…RMBライブラリ（作成したマップの保存に使用します）
  • TXT:MACRO.INI…マクロペンの内容を保存します
  • DAT:SAMPLE.RMB…サンプルデータ
