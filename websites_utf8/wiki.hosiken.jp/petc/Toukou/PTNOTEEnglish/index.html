An summary

"PTNOTE second" is simple and useful.

"PTNOTE second" is draw software.
It's like a notebook.

You choice a brush and draw illust.

When you push the START button,command mode is opening.

command mode can some setting.

When you push a "START" button, command mode will open.

When you push a "L" or "R" button, you can change colors.

If you want a new color, let's go writing comments.

Let's use the PTNOTE second!!
