2017年02月11日投稿 / 投稿者 : celeronist / タグ : 投稿プログラム OS ツール QRコード 初代対応? OSもどき OSモドキ

概要

４年くらい前に作ったOSモドキです。投稿したいなぁ〜って思ってたらいろいろ諸事情あっていつの間にか４年もたってしまいました。

システム構成

LIFEOSのシステムは主にシステムフォント、疑似電源ボタン用フォント、システムプログラムで構成されています。 別途でID、パスワード、その他諸々セーブデータが後々やってけば保存されます。

プログラムについて

OSモドキです。 デスクトップのメニューを開く場合は、虫眼鏡のキーを入れれば出てきます。 壁紙でデスクバーが見えない時の為にデスクトップでA,B,X,Yのボタンでバーのカラー配色を白,黒,赤,青の４色から定義できます（セーブ非対応）

内蔵（？）アプリケーション
  • BGMプレーヤー（プリセットのみ、OS内のアプリならバックで再生できます）
  • イメージビューア（GRPを読みだして鑑賞したり送ったりします、それだけ。こちらはカラー定義が出来ません。）
  • ワイヤレスモード（ファイルを送信、受信するモードです。当然ですが無線をONにする必要があります。）
  • テキストメモ（どっかのプログラムから拝借したような気がします。グラフィックの機能を応用した簡易的なメモ帳です。）　
  • 計算機（クッソ使いづらいと思われます。なるべく電卓のような流れにしたいと努力をしたような記憶はありますが。）
  • ペインター（超超簡易なペイントツールです。ほぼメモ代わりにどうぞ）
  • ロードファイル（LIFEOSを終了してほかのプログラムに切り替えます）
    • （LIFEOSの作業が一旦完全に終了してキャラ周り、BGMを完全にストップして指定したプログラムを起動します。）
    • （指定したプログラムが参照されない場合はOS内でエラーを返してデスクトップに戻りますが、BGMは停止状態になります。）
    • （補足ですが起動したプログラムに変数を入れた状態でこのOSを起動するとクイックでデスクトップに戻る機能を入れシムレースかつログインせずにできるようにしています、確か。解析してほかのプログラムで使ってみてください。）
  • コントロールパネル
    • 　いろいろ設定できます。

諸注意

  1. BGMプレーヤーにおいて、ボリューム初期値が０に設定されています。直す気があれば直せたかもしれませんが2013年製を守りたいのでそのまま放置してます。コントロールパネルで上げてから聞いてください。
  2. 一応穴は結構ふさいだ記憶はありますが多分ファイル周りで不正な入力をすると落ちるかもしれません。
  3. アップデート用にコントロールパネルで用意していますが。多分更新はしないと思います。勝手に弄って上げてくれればうれしいです。
  4. 追記欄（アップデートetc.）
    1. 現在特になし
    2. DSiのシェル交換に失敗して現在修正できない状態になりました・・・・・固定箇所のバグ等は手数かけますがスレにて返答したいと思います
    3. twitterの方で質問・要望なんかを受け付けたいと思います。IDは＠celeronable_ (2017年２月現在。)フォローは人を選ぶ体質なのであんまりしなくて結構です。速報とか上記の事とかに関してはリストに入れる程度でお願いします・・・。
    4. 念のために有志方々がアップデート、引き継ぎ等々をしてくれる場合は連絡願います。（こちらでもリンク貼り付けなどのなんらかの事をしたいとおもいますので・・・・）
