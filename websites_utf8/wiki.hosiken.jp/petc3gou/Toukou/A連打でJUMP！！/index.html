遊び方

遊び方を、簡単に説明します。

  • 1.「PLAY GAME」を選択
  • 2.「GO！」と同時に[Aボタン]を「連打」！
  • 3.どんどんスコアを延ばそう！

MENUについて

  • 「PALY GAME」
  • 「SCORE」
  • 「EXIT」
  • 「SET UP」
  • この4つについて説明します。

PLAY GAME

+  「PLAY GAME」について
  • 「GO！」と同時に[Aボタン]を “30秒間” 「連打」します。
  • 上画面の下の ゲージ が溜まれば溜まるほど、
    赤毛君がJUMPします。
  • 正直、期待以上に全然跳ばないです…
    ごめんなさい…

SCORE

+  「SCORE」について
  • これまでに出したスコアを5位まで
    見ることが出来ます。
  • 友達と競って見るのも良いですね。

EXIT

+  「EXIT」について
  • ゲームを終了します。
  • 保存していないスコアデータは終了すると消えてしまいます。
    気をつけてください。

SET UP

+  「SET UP」について
  • 「SET UP」には、2つの機能があります。
SAVE SCORE
+  「SAVA SCORE」について
  • スコアデータを保存します。
  • 一度上書きしたスコアデータは前のスコアデータを削除してしまいます。
    気をつけてください。
DELETE SCORE
+  「DELETE SCORE」について
  • 保存していたスコアデータを削除します。
  • 一度削除したスコアデータは元に戻りません。
    気をつけてください。
